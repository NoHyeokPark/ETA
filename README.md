# [약속 도우미 앱  - ETA]
> ETA - 'Estimated Time of Arrival', 도착 예정 시간을 공유하고 약속에 스트레스를 덜어내세요!
<br>
<br>

## 프로젝트 소개

- 지도로 약속 장소까지 얼마나 걸리는지 확인후에
- 알람을 맞춰서 늦지 않게 출발하고
- 실시간 위치 공유로 얼마나 걸리는지 확인해보세요  
- 아직 출발하지 않은 친구가 늦을거 같다면 제촉 메세지를 보내보세요
